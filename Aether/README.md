# Aether – Vision-Based 8 Ball Pool AI Agent

## Overview
Aether is a research-oriented autonomous agent that plays 8 Ball Pool on Android using only screen pixels as input and accessibility-service touch injection as output. No game memory is accessed. It combines computer vision, deep imitation learning, physics prediction, and human-like motor simulation.

## Features
- Screen capture via MediaProjection (no root)
- Ball detection using TensorFlow Lite
- Cue stick tracking with optical flow
- Physics trajectory prediction
- Policy network via imitation learning
- Humanized touch generation
- Modular C++17 codebase
